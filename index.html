Telegram Bot

This is a simple web interface for the Telegram bot. It helps to interact with the bot directly from Telegram. Below are the details and how to use the bot effectively.

Your Info

To get started, send /start or /help command to the bot in Telegram.

Compatible Files

  • .nm (NetMod Syna)
  • .tnl (OpenTunnel)
  • .ziv (Zivpn Tunnel)
  • .pb (PB Tunnel)
  • .sks (SocksHTTP)

Important

Remember, only authorized users can use this bot. You'll get a confirmation if your ID is approved.

Available Commands

Use /check_status to check if your ID is allowed or not.
